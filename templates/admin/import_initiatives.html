Import Initiatives from CSV
How it works:
  • Upload a CSV file with initiative data
  • If member email doesn't exist, a new account is created automatically
  • New accounts are auto-approved and will receive an email notification
  • Duplicate initiatives (same title by same user) are skipped
  • You can optionally auto-generate AI titles/descriptions
Max file size: 16MB
CSV Format Specification
Column Required Description
emailYesMember email (used to identify or create account)
nameYesFull name of member
organizationYesOrganization name
stakeholder_typeYesOne of: Government, NGO / Civil Society, Development Partner / Donor, Academic / Research, UN Agency, Private Sector
countryYesCountry name
initiative_titleYesTitle of the initiative
initiative_contentYesFull description/content
ai_titleNoPre-generated catchy title (leave blank for auto)
ai_descriptionNoPre-generated short description (leave blank for auto)
tagsNoComma-separated tags
is_publishedNotrue/false (default: true)
