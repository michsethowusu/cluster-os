Import Initiatives from CSV
How it works:
  • Upload a CSV — you'll see a preview before anything is saved
  • Emails must belong to existing approved members
  • If an email is unknown you can choose to create the member account first
  • All initiatives go into the approval queue — nothing is published automatically
  • Members receive an email when their initiative is submitted and again when it's published
  • Duplicate initiatives (same title, same user) are skipped
Max file size: 16MB
CSV Format Specification
ColumnRequiredDescription
emailYesMust match an existing approved member (or be created)
nameYes *Full name (used only when creating a new member)
organizationYes *Organization (used only when creating a new member)
stakeholder_typeYes *See valid types on the right (used only when creating)
countryYes *Country (used only when creating a new member)
initiative_titleYesTitle of the initiative
initiative_contentYesFull description / content (Markdown supported)
short_descriptionNoMax 300 chars summary
tagsNoComma-separated tags

* Required only when the email doesn't belong to an existing member.
