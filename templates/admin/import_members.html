Import or Invite Members via CSV
Import mode — adds members directly:
  • Upload a CSV file with member data
  • Required columns: email, name, organization, stakeholder_type, country
  • If a member already exists, it will be skipped
  • Imported members will be auto-approved and sent a welcome email
  • Imported members will be prompted to add projects on their first login
Invite mode — sends an invitation email only:
  • No accounts are created
  • Each person receives an email inviting them to register on the platform
  • People who already have an account are skipped
  • Only email and name columns are required in invite mode
Custom message mode — sends your own notification:
  • No accounts are created
  • Each person receives your custom email using the platform notification template
  • An unsubscribe link is automatically appended
  • Only email and name columns are required
Max file size: 16MB
When checked, no accounts are created. Recipients receive an email with a link to register themselves on the platform.
When checked, each person in the CSV receives a custom email using the platform's notification template. No accounts are created. Only email and name columns are required.
Plain text only. The platform's standard email header and an unsubscribe footer will be added automatically.
