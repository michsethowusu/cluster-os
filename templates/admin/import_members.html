Import or Invite Members via CSV
Import mode — adds members directly:
  • Upload a CSV file with member data
  • Required columns: email, name, organization, stakeholder_type, country
  • If a member already exists, it will be skipped
  • Imported members will be auto-approved and sent a welcome email
  • Imported members will be prompted to add projects on their first login
Invite Organisations — sends invitation email on behalf of their organisation:
  • No accounts are created
  • Each person receives an email inviting them to register on behalf of their organisation
  • Required columns: email, name, organization
Invite Individuals — sends invitation email to join in their own capacity:
  • No accounts are created
  • Each person receives an email inviting them to register as an individual member
  • Required columns: email, name
Custom message mode — sends your own notification:
  • No accounts are created
  • Each person receives your custom email using the platform notification template
  • An unsubscribe link is automatically appended
  • Only email and name columns are required
Event invitation mode — sends an event-specific invitation:
  • No accounts are created
  • Each person receives an email with the event details and a link to register
  • Emails are sent to everyone on the list regardless of membership status
  • Only email and name columns are required
Max file size: 16MB
